UpCap Android 테스트 빌드

휴대폰에서 바로 설치하고 확인하세요.

이 페이지는 항상 최신 debug APK로 연결됩니다. 개발 PC에 휴대폰을 연결하지 않아도 링크를 열어 설치하고 모델 다운로드 상태를 확인할 수 있습니다.

최신 APK 다운로드 릴리스 페이지 보기

설치 순서

  1. 1 최신 APK 다운로드위 버튼을 눌러 app-debug.apk를 받습니다.
  2. 2 알 수 없는 앱 설치 허용브라우저 또는 파일 앱에 설치 권한을 허용합니다.
  3. 3 앱 실행 후 모델 다운로드홈 화면의 AI 모델 준비에서 두 모델이 준비됨이 되는지 확인합니다.
  4. 4 짧은 영상으로 먼저 테스트자막 생성과 화질 개선을 각각 10초 안팎 영상으로 확인합니다.

PC에서 보는 중이면

휴대폰 카메라로 QR을 스캔하면 이 설치 페이지가 열립니다.

테스트 전 확인할 점

이 APK는 debug 빌드입니다. 처음 실행할 때 모델 다운로드가 필요하며, 네트워크 상태에 따라 시간이 걸릴 수 있습니다. 앱 안에서 모델 상태가 준비됨으로 표시되기 전에는 처리 시작이 막히도록 되어 있습니다.
